## **Credit/Resource Allowances**

### **Snov.io Starter Credit Pool**

| Feature | Monthly Allowance | Notes |
|---------|-------------------|-------|
| **Total credits** | 1,000 | Shared across all features |
| **Email finding** | 1,000 credits | 1 credit = 1 email found |
| **Email verification** | Shared pool (1,000) | 1 credit = 1 email verified |
| **Domain search** | Shared pool (1,000) | 1 credit per search |
| **LinkedIn search** | Shared pool (1,000) | 1 credit per search |
| **Database search** | Shared pool (1,000) | 1 credit per search |
| **Bulk operations** | Limited by total | Max 5,000 bulk emails (but only 1,000 credits) |

**Credit Model:** 
- 1 credit = 1 email found/verified (or 0 if not found)
- All finder, verifier, and outreach actions draw from **same 1,000 credit pool**
- No rollover to next month

### **Saleshandy Starter — No Credit System**

| Feature | Monthly Allowance | Notes |
|---------|-------------------|-------|
| **Flat-fee model** | $25/mo | No credits — all features unlimited once subscribed |
| **Email finding** | Not included | Must use external tool (SalesQL, Prospeo, Apollo, etc.) |
| **Email verification** | Not included | Must use external tool or TrulyInbox |
| **Active prospects** | 2,000 | Binding constraint (contacts in live sequences) |
| **Email sends** | 6,000/mo | Unlimited sends per contact |
| **Sequences** | Unlimited | Multi-step sequences included |
| **A/B testing** | Up to 26 variants | Unlimited tests |

**Credit Model:** 
- Flat subscription — no credit consumption
- Pay once, use all features within limits

## **What Each Plan Is Best For**

### **Snov.io Starter ($39/mo)**
✓ **All-in-one for small teams**
- Email finding + verification built-in
- Warm-up included
- Sequences included
- Personalization included
- No need for external tools

⚠️ **Constraint:** 1,000 monthly credits shared across all features

**Best if:** You need reverse email lookup + warm-up + sequences in one tool

---

### **Saleshandy Starter ($25/mo)**
✓ **Email-sending specialist (cheap entry point)**
- Cheapest option ($25/mo)
- Unlimited email accounts
- Best A/B testing (26 variants)
- Good for high-volume sending

✗ **Gaps:** No email finding or verification — must pair with external tools

**Best if:** You already have verified email lists and only need warm-up + sending
