## Quick Reference

| Task | Use |
|---|---|
| Quick question or research | Claude Chat (claude.ai) |
| Working on this wiki | Claude Code inside `Obsidian Vault/` |
| Working on a code project | Claude Code inside that project folder |
| Persistent instructions for a project | `CLAUDE.md` in the project folder |
| Personal preferences everywhere | `~/.claude/CLAUDE.md` |
