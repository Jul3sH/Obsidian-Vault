When listing goals (or any content grouped by workstream — e.g. "list all
current goals"), always present the six workstreams in the priority order
defined in [[workstream-framework]]: **Wellbeing, Relationships, Finance,
Career, Performance, Personal**. Do not default to alphabetical order.

**Why:** Julian confirmed this hierarchy reflects a genuine flow/priority —
"if I've got to prioritize one goal over another." Alphabetical ordering
(Career, Finance, Performance...) obscures that priority.

**How to apply:** Any time goals, OKRs, SMART goals, or workstream summaries
are listed across multiple workstreams, group and order them
Wellbeing → Relationships → Finance → Career → Performance → Personal.
This is a display/ordering convention only — it does not mean every
Wellbeing goal outranks every Relationships goal individually; see
[[prioritization-framework]] for how broad cross-goal ranking works.
