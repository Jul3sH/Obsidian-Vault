## How to read this table

- **ID** — short reference used by the [[habits-register]] to link habits to goals
- **Objective** — OKR Objective (Career and Performance only); "—" for SMART-goal workstreams
- **Goal / KR** — the specific SMART goal or Key Result; full text in the workstream file
- **Target** — deadline or quarter
- **Projects** — wiki Projects currently driving this goal; see [[project-priorities]] for full detail
- **Habits** — habits from [[habits-register]] that compound toward this goal; populate as habits are defined
- **Status** — in progress / upcoming / done / paused
