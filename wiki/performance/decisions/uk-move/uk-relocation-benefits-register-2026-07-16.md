> Current upside surface for the [[uk-relocation-decision]]. This is the sibling to the risk register: risks ask "what can go wrong"; this file asks "what does each option positively deliver if it works?"

## Key Takeaways

- The benefits are scattered across old BRAINED docs, the financial model, job-market work, schooling notes, TTI logs, and the Malvern-permanent feasibility model.
- The old [[benefits]] file is still useful, but it uses the original four BRAINED scenarios. It is not current enough for the live option set.
- London-direct has the broadest life-design upside: one school transition, home nightly with Sophia, London social life, and the strongest career-market base.
- Malvern interim has the strongest search-phase runway upside: much lower burn and preserved optionality, but only if the human and tenancy risks are controlled.
- Stay-HK via a real TTI permanent role has the highest clean financial and continuity upside, but only if it firms up before the school deadline.
- Permanent Malvern semi-retirement is a real fallback benefit set, not a primary plan: ease, Sophia presence, and low burn, at the cost of career and pension-buffer consumption.
