Situation: Coming across a new finding whilst studying or doing a project, and then excessively bleeting on about it and repeat myself. I have done this in the past. 

Root cause: I think this gets back to my personality type requiring approval, so if I don't get the response I want, thus giving me that approval, I tend to keep repeating myself thinking that people haven't got it, or else they would be as excited as I am. 

Lesson Learnt: Tell everyone, document it as a recommendation so that there is an audit trail, and move on. If I don;t get the required response, think to myself "is it that it just isn;t important to the audeince, or is it that I haven't comunicated it effectively. If the former, leave it, it hte latter, try a different approach
